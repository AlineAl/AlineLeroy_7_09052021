## FRONT-END

1) Pour faire fonctionner la partie front-end, il faut procéder à l'installation de Node.js, MySQL et Vue.js soit à l'aide des pack `npm` ou `yarn`.

2) Taper dans le terminal la commande `yarn serve`

3) Aller sur `http://localhost:8080/`

## BACK-END

1) Pour se connecter à la base de donnée, aller dans le dossier `backend` puis `config` et cliquer sur le fichier `config.json`. Modifier le username et le password

2) Taper dans le terminal `sequelize db:create` puis `sequelize db:migrate`

3) Effectuer dans le terminal un `npm install` ou si vous utilisez yarn, un `yarn install`

4) Taper dans le terminal la commande `node server`

5)  Aller sur `http://localhost:3000/`
